## 기능 요구 사항
로또 구입 금액(1장 1000원)을 입력하면 구입 금액에 해당하는 로또를 자동으로 발급해야 한다.
- 로또 번호 생성
  - 1~45 번호 섞기
  - 앞에서 6개 선택
  - 선택한 번호 정렬
- 로또 번호 하나의 역할을 하는 포장 클래스 구현
  - 범위 검증
- 로또 1게임의 역할을 하는 일급 컬렉션 구현
  - 개수 검증
  - 중복 검증
  - 당첨 번호 몇 개와 일치하는지 확인
    - 하나의 당첨 번호와 일치하는 번호가 있는지 확인
- 로또 1장의 역할을 하는 일급 컬렉션 구현
  - 당첨 번호 일치 개수별 게임 수 확인
- 로또 자동 판매기 역할을 하는 클래스 구현
  - 로또 티켓 판매
  - 로또 티켓 당첨 확인
- UI
  - 구매 금액 입력
  - 당첨 번호 입력
  - 당첨 통계 출력
