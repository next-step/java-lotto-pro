## 요구사항 정리

* 로또를 발급한다.
    * [X] 로또 번호는 1부터 45의 숫자만 허용한다.
    * [X] 로또는 6개의 번호를 가질 수 있다.
    * [ ] 6개의 로또 번호를 자동 생성한다.
    * [ ] 로또 1장의 가격은 1000원이며, 입력받은 금액의 최대로 로또를 발급한다.
* 당첨 번호와 3개이상의 번호가 일치하면 돈을 받는다.
    * [ ] 3개 일치 시 5000원
    * [ ] 4개 일치 시 50000원
    * [ ] 5개 일치 시 1500000원
    * [ ] 6개 일치 시 2000000000원
* 당첨 통계를 볼 수 있다
    * [ ] 당첨 금액별 개수를 볼 수 있다.
    * [ ] 총 수익률을 계산해서 보여준다.
