## 요구사항

### 로또 가격

- [x] 1000원

### 로또 숫자

- [x] 1 -45 범위의 숫자

### 로또

- [x] 중복 되지 않은 로또 숫자를 6개 가짐
- [x] 가격을 가짐
- [x] 번호 순 정렬
- [x] 당첨 결과 반환

### 당첨 결과

- [x] 로또 번호와 당첨 번호를 이용해 맞은 개수 계산
- [x] 맞은 개수를 이용해 당첨 금액 계산

### 로또 랜덤 생성기

- [x] 랜덤한 로또 하나를 생성한다

### 로또 자동 발급기

- [x] 구입 금액에 따라 해당 개수 만큼 로또를 발급

### 당첨 통계기

- [ ] 당첨 번호를 입력 받는다
- [ ] 각 순위 로또 개수 계산
- [ ] 수익률 계산
