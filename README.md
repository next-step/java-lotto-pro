# 로또
## 3단계 미션 기능 목록
* 구매 금액을 입력받는다.
  * 구매 금액 객체 생성
  * 입력값 유효성 검증
  * 구입 가능한 로또 티켓 수량 계산


* 로또 티켓을 생성한다.
  * 로또 번호들 객체 생성
    * 입력값 유효성 검증
  * 로또 티켓 객체 생성
  * 자동 로또 번호 발급기 객체 생성
  * 로또 티켓들 객체 생성


* 당첨 번호를 입력받는다.


* 로또 티켓별 상금을 판별한다.
  * 상금 유형 객체를 생성한다.
  * 로또 티켓별 어떤 상금 유형을 가지는지 확인한다.


* 당첨 결과를 출력한다.
  * 당첨 결과 객체 생성
  * 당첨 유형별 통계 계산
    * 당첨 통계 객체 생성
  * 수익률 계산


* 입력 및 결과 안내 메시지를 출력한다.
